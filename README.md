The Neobank Audit Engine is a Python-based utility designed to streamline the auditing process for financial transaction data. The project automates data processing and verification, ensuring accuracy and compliance in handling complex banking datasets.

Core Functionality
Data Analysis: Leverages the pandas library to perform high-efficiency auditing and manipulation of large-scale transactional data.

Database Connectivity: Implements secure mysql-connector-python integration to bridge the gap between backend logic and transactional databases.

Audit Automation: Provides a structured environment for verifying transaction logs and identifying discrepancies within the neobank ecosystem.

Technical Stack
Language: Python

Data Handling: pandas

Database Driver: mysql-connector-python

Frontend/Dashboard: Streamlit
